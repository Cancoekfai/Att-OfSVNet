# Att-OfSVNet

Code to run Offline Signature Verification from my research Offline Signature Verification Based on Att-OfSVNet on a Large Chinese Signature Dataset.


# Installation

## Basic

- `Python` >= 3.6

## Modules

- `matplotlib`
- `numpy`
- `pandas`
- `scikit-learn`
- `torch` >= 1.8.1
- `torchvision` >= 0.9.1
- `tqdm`

```shell
pip install -r requirements.txt
```
